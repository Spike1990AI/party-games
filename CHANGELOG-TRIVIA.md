# Changelog - Trivia Showdown

All notable changes to the Trivia Showdown game will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-26

### Added
- Initial release for 4 players
- 10 questions per game
- Speed-based scoring (faster answers win more points)
- Real-time multiplayer via Firebase
- Multiple choice question format
