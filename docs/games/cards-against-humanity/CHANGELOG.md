# Changelog - Cards Against Humanity

All notable changes to the Cards Against Humanity game will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-26

### Added
- Initial release for 3-8 players
- Card Czar rotation system
- Configurable winning score (5, 7, or 10 points)
- Real-time multiplayer via Firebase
- Adult humor card deck
- Black card (prompts) and white card (answers) system
- Anonymous submission and judging
- Round winner display with winning card combo
- Game over screen with final scores
